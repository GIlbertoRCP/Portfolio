      <h2 class="text-2xl font-display font-extrabold text-white mb-3">Architectural Implementation</h2>
      <p>
        The backend code leverages several key object-oriented structures:
      </p>
      
      <ul class="list-disc pl-6 space-y-2 mt-2">
        <li><strong>Base Class and Inheritance:</strong> A base class <code>Product</code> defines common properties such as Name and Price, alongside a virtual method <code>CalculateShipping()</code>. Subclasses (<code>PhysicalProduct</code>, <code>DigitalProduct</code>, and <code>SubscriptionProduct</code>) inherit from the base and extend specific behavior.</li>
        <li><strong>Polymorphism in calculations:</strong> Subclasses override the virtual shipping method. For instance, <code>PhysicalProduct</code> computes weight-based shipping fees ($5.00 per kg), whereas <code>DigitalProduct</code> and <code>SubscriptionProduct</code> return zero immediately since they do not require physical freight.</li>
        <li><strong>C# Generic Collections:</strong> Implemented a <code>List&lt;Product&gt;</code> to manage the shopping cart instance. This generic container avoids box-unbox performance penalties and exposes LINQ methods for real-time calculations.</li>
      </ul>
    </div>

    <div>
      <h2 class="text-2xl font-display font-extrabold text-white mb-3">OOP Structure Snippet</h2>
      <p>
        Below is a simplified representation of the C# domain model implemented for the catalog:
      </p>
      
      <pre class="bg-slate-950/80 border border-slate-850 p-4 rounded-xl text-xs text-slate-300 font-mono overflow-x-auto leading-relaxed">
{`public abstract class Product
{
    public string Name { get; set; }
    public double Price { get; set; }

    protected Product(string name, double price)
    {
        Name = name;
        Price = price;
    }

    public abstract double CalculateShipping();
}

public class PhysicalProduct : Product
{
    public double Weight { get; set; }

    public PhysicalProduct(string name, double price, double weight) 
        : base(name, price)
    {
        Weight = weight;
    }

    public override double CalculateShipping()
    {
        return Weight * 5.00; // $5.00 per kg
    }
}`}
      </pre>
